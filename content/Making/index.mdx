# Making

## Technologies

This site was created using [Canopy IIIF](https://canopy-iiif.github.io/docs/about), an open-source static site generator designed for fast creation, contextualization, and customization of a discovery-focused digital scholarship and collections website using IIIF APIs.  See the [Create a Canopy Project guide](https://canopy-iiif.github.io/docs/get-started) for detailed step-by-step instructions to create your own site.

Maps were created using [ArcGIS](https://www.arcgis.com/index.html).

3D images were created using [Blender](https://www.blender.org/) and [SketchFab](https://sketchfab.com/feed).
